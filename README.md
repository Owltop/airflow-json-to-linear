# JSON to Linear Structure - ETL Pipeline

## Описание проекта
Домашее задание 1 по ETL-процессам. Проект демонстрирует преобразование вложенной JSON структуры в линейную таблицу PostgreSQL с использованием Apache Airflow.

## Запуск проекта

```bash
docker-compose up -d
```
